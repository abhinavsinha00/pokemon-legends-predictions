# ⚡ Pokemon Legendary Prediction

A Machine Learning web app that predicts whether a Pokémon is Legendary or Not Legendary using Pokémon stats.

## 🚀 Features

- Clean Streamlit UI
- Real-time prediction
- Probability score output
- Stats-based classification model

## 🛠️ Tech Stack

- Python
- Pandas
- Scikit-learn
- Streamlit

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
